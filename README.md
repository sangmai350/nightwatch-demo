# nightwatch-demo

# BDD base project: Nightwatch + Cucumber + Javascript
Basic example of using Nightwatch with Cucumber (BDD). This `Gherkin` example includes:
- Basic Scenario
- Scenario Outline
- Tagged tests

# Get started

## Installation

```bash
npm install
```

# How to run the tests
